## 一、底部导航栏

| 测试点 | 操作 | 预期结果 | 问题记录 |

## 二、记账页（AddBill）

### 类型切换

| 测试点 | 操作 | 预期结果 | 问题记录 |
|--------|------|----------|----------|
| 切换为收入 | 点击「收入」 | 顶部背景变绿，分类宫格显示：工资 / 兼职 / 理财 / 红包 / 其他 | 没有问题 |

### 金额输入

| 测试点 | 操作 | 预期结果 | 问题记录 |

## 六、端到端流程

| 流程 | 步骤 | 验证点 | 问题记录 |
|------|------|--------|----------|
| 完整记账流程 | 记账页填写并保存 → 切换到账单页 | 新账单出现在列表，月汇总数字正确 | |
| 删除后刷新 | 删除一条账单 | 当日分组金额更新，月汇总更新，分组消失（若当日仅此一条） | |
| 跨月数据隔离 | 添加不同月份账单，用 ‹ › 切换 | 各月只显示本月数据，互不影响 | |
| 统计与账单联动 | 在记账页新增支出后进入统计页 | 饼图、柱状图、分类明细数据同步更新 | |
| 预算全流程 | 1. 设置页为「餐饮」设置预算 100 元并保存<br>2. 记账页记一笔餐饮 90 元<br>3. 再记一笔餐饮 20 元 | 第二笔保存后出现超预算 Toast「「餐饮」本月已超预算 ¥10.00」 | |
