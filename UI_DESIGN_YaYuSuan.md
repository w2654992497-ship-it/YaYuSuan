# 鸭鱼算 (YaYuSuan) UI/UX 设计规范文档

## 1. 设计理念 (Design Concept)
**主题：“冰川水与暖阳鸭” (Ice Water & Warm Sun Duck)**
- **视觉隐喻**：水、深海、浮力、扁平、克制的理智 + 鸭子、明黄、灵动、局部的警醒。
- **设计心智**：财务记账需要**严谨、冷静、不杂乱**，而“压预算”的谐音梗带来了**趣味性**。界面 90% 的空间留白并使用同色系（深水青、冰蓝灰）以降低认知负担，10% 用于关键交互（鸭嘴黄、祖母绿、玫瑰红）。

## 2. 色彩系统 (Color System)
| 颜色作用 | 色值 | 说明 |
| --- | --- | --- |
| **页面底色 (Background)** | `#F0F4F8` | 冰蓝色，极浅蓝灰，提供舒适的阅读底色，代替纯白。 |
| **主品牌色 (Primary)** | `#2C7A7B` | 深水青，饱和度低，理智且深邃，用于顶部导航、主要激活状态。 |
| **高亮撞色 (Accent/Duck)** | `#F59E0B` | 鸭嘴橙金，仅用于悬浮记账按钮 (FAB) 及关键进度/越界警示。 |
| **前景色 (Surface/Card)** | `#FFFFFF` | 纯白色，承载具体信息的卡片底色。 |
| **主要文字 (Text Primary)** | `#1A202C` | 极深蓝灰，避免死黑，与体系保持微弱统一。 |
| **次要文字 (Text Sec)** | `#718096` | 中级蓝灰，用于时间、备注、非激活 Tab。 |
| **收入状态 (Income)** | `#059669` | 祖母绿，低明度的绿，代表生机与向上增值。 |
| **支出状态 (Expense)** | `#E11D48` | 玫瑰红，克制的红色，代表预警与流失。 |
| **分割线 (Divider)** | `#E2E8F0` | 用极浅灰色界定边界，不喧宾夺主。 |

## 3. 造型与结构 (Shape & Layout)
- **水滴圆角 (Water Drop Radius)**：
  - 核心包裹卡片 (Card) 统一采用 `16vp` 或 `24vp` 大圆角，抹除直角带来的生硬感。
  - 按钮 (Button) 呈胶囊状或大圆角形态。
- **无边界设计 (Borderless)**：
  - 尽量去除物理线条 (Border)，依靠 `#F0F4F8` 的底色与 `#FFFFFF` 卡片的色差形成自然的区块隔离，必要时增加极轻微的弥散阴影 `rgba(44, 122, 123, 0.05)`。
- **沉浸式布局 (Immersive)**：
  - 导航栏与状态栏背景融为一体，延伸可视范围。

## 4. 图标系统 (Iconography)
- **风格**：扁平化 (Flat)、微圆润 (Rounded)、双色调 (Duotone 概念，主要用纯色配合透明度)。
- **吉祥物隐喻 (鸭与鱼)**：
  - **餐饮**：小鸭子吃米/鱼骨头。
  - **交通**：鸭鸭骑滑板车。
  - **娱乐**：吐泡泡的鱼或游戏机。
  - **新增记账 (FAB)**：以显眼的 `鸭嘴橙金` 为底的“➕”或鸭鱼符号嵌套。

## 5. 交互与动效 (Interaction & Animation)
- **果冻弹簧回弹 (Spring Animation)**：
  - 底部 Tab 和悬浮大黄鸭按钮点击时，采用缩放 (`Scale 0.95 -> 1.05`) + `Curve.Spring` 动效。
- **波纹反馈 (Ripple)**：
  - 每笔账单条目 (`BillItem`) 点击时应有轻微的点按波纹或背景变色反馈。
- **滑动删除 (Swipe to Delete)**：
  - 账单列表右滑出现的操作区背景使用 `#E11D48`，图标居中，伴随阻尼滑动。

## 6. 各页面重构指导
1. **账单列表 (BillList)**：去线化，按月/日聚合的数据块使用大圆角白色卡片，内部账单条边距拉开，金额字体采用 `Bold`。
2. **记账录入 (AddBill)**：顶部采用深水青背景，金额输入极简超大字号；分类图标做成大圆角网格，选中时出现橙金边框或同色系底色。
3. **图表统计 (Statistics)**：图表采用祖母绿与玫瑰红的低饱和配色，图例圆整，月份切换做成胶囊按钮。
4. **我的/设置 (Settings)**：标准设置项卡片化列表，预算超支提醒加入橙金色进度条。
