# 小金鱼记账风格验证图 V2

本文件夹用于验证新版视觉方向，当前包含 2 张优先判断风格的页面：

- `validation_v2_01_home.png`：账单首页，偏信息效率。重点验证月度摘要、账单列表、筛选按钮和底部导航的可实现性。
- `validation_v2_02_add_bill.png`：记账页，偏品牌表达。重点验证顶部品牌区域、金额输入、分类网格和保存按钮。

## 当前设计判断

新版方向保留“金鱼”品牌，但不做幼稚卡通：

- 背景使用浅水蓝，整体保持清爽。
- 核心摘要卡使用深青水色，提高账务信息的视觉重量。
- 金鱼元素主要出现在背景、水波、鱼尾曲线、气泡和空状态中。
- 页面主体卡片保持干净，避免每个组件都装饰化。

## 实现约束

交互控件不要复杂化，方便后续 ArkUI 实现：

- 按钮只使用纯色、白底描边、危险描边三类。
- 图标选中态只改底色和图标色，不使用复杂光效、纹理、双层气泡或渐变描边。
- Tab 选中态只改变图标和文字颜色。
- 鱼鳞纹理、水波、金鱼剪影只用于背景、摘要卡、空状态、弹窗角落等非交互区域。
