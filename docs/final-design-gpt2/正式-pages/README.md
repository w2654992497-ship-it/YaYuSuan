# 小金鱼记账正式设计页

本目录用于存放“小金鱼记账”的正式页面设计图。当前目录内已有 10 张页面/弹窗图，文件均为独立页面设计稿，不是从总览图裁剪。

| 文件 | 页面 |
| --- | --- |
| `01_home.png` | 账单首页 |
| `02_add_bill.png` | 记账页 |
| `03_search.png` | 搜索 / 筛选页 |
| `04_settings_budget.png` | 设置 / 预算页 |
| `05_calendar.png` | 日历页 |
| `06_statistics.png` | 统计页 |
| `07_category_manage_expense.png` | 分类管理：支出 |
| `08_category_manage_income.png` | 分类管理：收入 |
| `09_edit_category_modal.png` | 编辑分类弹窗 |
| `10_edit_icons_page.png` | 编辑记账页图标页 |

后续重设视觉方向时，以 `STYLE_REDESIGN.md` 为准，不需要完全沿用当前图片里的黄色顶部、大面积鱼形装饰和原始页面结构。
