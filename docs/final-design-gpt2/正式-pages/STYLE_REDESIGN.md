# 小金鱼记账新风格语言：金鱼游记风

## 1. 目标用户

主要受众：学生群体。

设计目标：

- 年轻、轻松，但不是儿童化。
- 有明确品牌记忆点，但不牺牲记账效率。
- 看起来比传统记账 App 更有活力，比游戏化 App 更克制。
- 让用户感觉“记账不是负担，而是在记录自己的生活流动”。

一句话方向：

**像一条小金鱼在清水里游过，把每天的收支留下清楚、轻快的轨迹。**

---

## 2. 风格关键词

| 关键词 | 说明 |
| --- | --- |
| 金鱼感 | 不只是一条鱼插画，而是一套鱼尾、水波、气泡、鱼鳞纹理系统。 |
| 年轻化 | 更轻快、更明亮、更像学生日常工具，不做商务财务感。 |
| 清水感 | 保持浅水蓝、留白、低压感，避免传统金色招财风。 |
| 有余感 | 品牌心智不是“花钱焦虑”，而是“看清余量”。 |
| 轻工具 | 功能必须清晰，所有装饰都服务于信息层级。 |

## 3. 视觉隐喻系统

### 3.1 金鱼剪影

使用方式：

- 首页、统计页、空状态可以出现完整小金鱼。
- 其他页面只使用局部剪影：鱼尾、背鳍、游动弧线。
- 金鱼造型要几何化、扁平化、年轻化。

避免：

- 大眼卖萌表情。
- 复杂拟真鳞片。
- 每个页面都放一条完整金鱼。
- 像儿童贴纸一样到处漂浮。

### 3.2 鱼尾曲线

鱼尾曲线是核心装饰语言。

用途：

- 页面顶部背景的水波尾迹。
- 摘要卡片内部的分割线。
- 按钮 hover/选中态的尾巴形高光。
- 弹窗顶部把手下方的轻水波。

视觉要求：

- 曲线要轻、细、透明度低。
- 不要像海浪大图案。
- 一页最多 2-3 处明显曲线。

### 3.3 水波轨迹

水波轨迹表达“钱的流动”。

用途：

- 首页账单列表分组之间的轻分割。
- 统计页图表背景。
- 日历页选中日期周围的小波纹。
- 搜索结果命中高亮的底纹。

### 3.4 气泡标签

气泡是标签和状态的视觉来源。

用途：

- `全部 / 收入 / 支出` 筛选标签。
- 分类管理里的“默认”标签。
- 搜索页里的筛选条件。
- 图标选择里的选中态。

气泡标签特征：

- 圆润胶囊。
- 可带轻描边。
- 选中态像“亮起的小泡泡”，不使用厚重按钮。

### 3.5 鱼鳞纹理

鱼鳞纹理只做微弱底纹。

用途：

- 分类图标圆底。
- 主要按钮的极浅纹理。
- App 图标背景。
- 空状态插图背景。

要求：

- 透明度低于 8%-12%。
- 不影响文字可读性。
- 不能铺满大面积正文区域。

## 4. 新颜色系统

### 4.1 主色板

| 名称 | 色值 | 用途 |
| --- | --- | --- |
| 清水浅蓝 | `#EAF8FA` | 页面背景、空状态背景 |
| 浅水蓝 | `#CFF3F2` | 顶部氛围、卡片浅渐变 |
| 深青水色 | `#0F6F73` | 摘要卡、重要结构块 |
| 深海墨蓝 | `#17323D` | 标题、金额、主要文字 |
| 金鱼橙 | `#FF8A3D` | 主按钮、品牌重点、选中态 |
| 金鱼高光 | `#FFD27A` | 高光、浅选中底、气泡光 |
| 珊瑚支出 | `#F45D6C` | 支出金额、警示 |
| 青绿收入 | `#35B977` | 收入金额 |
| 雾灰蓝 | `#8FA1AF` | 辅助文字、未选中图标 |
| 水线灰 | `#E2EEF0` | 分割线、卡片边框 |

### 4.2 配色比例

- 背景清水浅蓝：55%
- 卡片白色：25%
- 深青水色：10%
- 金鱼橙 / 金鱼高光：7%
- 收入绿 / 支出红：3%

关键原则：

- 金鱼橙是品牌点睛，不是整页底色。
- 深青水色承载金额和统计，形成专业感。
- 清水浅蓝负责年轻、清爽、低压。

## 5. 页面结构语言

### 5.1 首页

目标：第一眼看到“这个月还剩多少”，第二眼看到钱花在哪里。

重构建议：

- 顶部使用浅水蓝背景，金鱼剪影或鱼尾水波作为局部装饰。
- 品牌标题可以更年轻：`小金鱼记账` + 小气泡搜索入口。
- 摘要卡使用深青水色，不再用大面积黄色。
- 摘要卡上可以有 1 条金鱼游动轨迹，金额必须保持最大。
- 筛选标签改成气泡标签。
- 账单项左侧分类图标使用鱼鳞圆底。

### 5.2 记账页

目标：快速记一笔，金额输入要强。

重构建议：

- 顶部做“水面输入区”，但高度不要过大。
- 支出/收入切换像两个气泡，选中态为金鱼橙。
- 金额输入区可以有鱼尾曲线作为下划线。
- 日期胶囊像小气泡。
- 分类网格使用鱼鳞纹理底，选中态像被气泡包住。
- 保存按钮可以使用“鱼尾按钮”：右侧有轻微尾巴曲线或高光。

### 5.3 搜索页

目标：快速找到账单，并清楚看到搜索结果总额。

重构建议：

- 搜索框是顶部主角，像一条长气泡。
- 筛选条件用气泡卡片，不要厚重表单。
- 结果汇总卡必须保留：收入、支出、净额。
- 命中字段用金鱼高光底纹。
- 底部可以有低透明水波，不影响结果阅读。

### 5.4 设置 / 预算页

目标：预算管理更像“学生生活预算面板”。

重构建议：

- 顶部加入轻量预算状态：本月预算使用情况。
- 分类预算不做表格，改成小卡片 + 水位进度条。
- 进度条可用“水位”隐喻：已用部分像流动水线。
- 导出 CSV 是低频操作，视觉降级。

### 5.5 日历页

目标：看每一天收支痕迹。

重构建议：

- 日期格子的记录点改成小气泡。
- 选中日期外圈像水波扩散。
- 有支出日显示珊瑚小点，有收入日显示绿色小点。
- 当日空状态可以用“漂浮账单纸 + 小气泡”，不一定放完整金鱼。

### 5.6 统计页

目标：看清钱流向哪里。

重构建议：

- 图表区域更专业，金鱼元素弱化为水波背景。
- 环形图可以像“鱼泡圆环”，但不要过粗。
- 分类排行条可以像小水流。
- 统计页应比其他页面更克制。

### 5.7 分类管理页

目标：管理分类像整理自己的生活标签。

重构建议：

- 支出/收入切换用气泡分段控件。
- 分类行卡片更轻，拖拽点像一串小气泡。
- 默认标签像小泡泡。
- 添加分类按钮可以是虚线鱼鳞边框。
- 页面顶部可以有鱼尾曲线，不要大金鱼。

### 5.8 编辑分类弹窗

目标：明确编辑名称、图标、颜色。

重构建议：

- 弹窗顶部有水波把手。
- 当前分类预览卡使用鱼鳞底纹。
- 图标选择区使用气泡网格。
- 颜色选择区像一排鱼鳞色盘。
- 确定按钮使用金鱼橙，取消按钮弱化。

### 5.9 图标编辑页

目标：批量调整分类图标，轻松直接。

重构建议：

- 分类图标改为“鱼鳞徽章”。
- 选中态是双层气泡圈。
- 底部添加、重置、完成要层级清楚。
- 页面留白可以更大，但要避免空洞。

## 6. 图标系统

### 6.1 分类图标

图标结构：

```text
浅水蓝圆底
  + 深青线性图标
  + 选中时只切换为金鱼橙圆底 + 白色图标
```

图标底色建议：

| 分类 | 色值 |
| --- | --- |
| 餐饮 | `#FF8A3D` |
| 交通 | `#4B9DEB` |
| 购物 | `#F45D8A` |
| 娱乐 | `#9B5DE5` |
| 住房 | `#22B8A7` |
| 医疗 | `#F25F5C` |
| 教育 | `#5E6CE6` |
| 其他 | `#7895A1` |

### 6.2 Tab 图标

- 未选中：雾灰蓝。
- 选中：只改为深海墨蓝，不加气泡、不加底纹。
- 中间新增按钮：金鱼橙圆按钮，不加复杂纹理。

### 6.3 交互控件实现规则

按钮和选中态必须简单，复杂视觉只放在背景、摘要卡、空状态和少量品牌插图中。

按钮规则：

- 主按钮：纯 `#FF8A3D`，白字。
- 次按钮：白底 + `#E2EEF0` 边框 + 深海墨蓝文字。
- 危险 / 重置按钮：白底 + `#F45D6C` 边框和文字。
- 禁用按钮：浅灰底 + 雾灰蓝文字。

选中态规则：

- 筛选标签选中：纯深青水色或金鱼橙背景 + 白字。
- 分类图标选中：只换圆底色和图标色，最多加 2px 外描边。
- Tab 选中：只改图标和文字颜色。
- 不使用发光、复杂阴影、鱼鳞纹理、渐变描边、双层气泡圈作为交互状态。

金鱼元素使用边界：

- 可以用于页面背景、摘要卡、空状态、弹窗顶部装饰。
- 不进入按钮、Tab、筛选标签、分类选中态的细节。
- 交互控件优先考虑 ArkUI 可实现性。

## 8. 验收标准

- 一眼能感受到“小金鱼”品牌，而不是通用清爽 App。
- 金鱼元素来自系统化设计语言，而不是到处贴鱼插画。
- 页面仍然清楚、可读、可实现。
- 学生用户会觉得轻快、亲近，不会觉得幼稚。
- 重点金额、按钮和筛选状态清晰。
- 每页金鱼元素不超过 3 种，避免装饰过载。
